## How to Install Python

```
1. https://www.anaconda.com/products/individual 접속

해당 사이트에서 다운로드 받아 설치하도록 한다.
파이썬 버전은 3.x 버전으로 다운 받아야 한다.
설치 시간이 좀 걸릴 수 있습니다.

2. 실행
3. Next
4. 동의
5. 중간쯤에 Just me (recommended)
6. 경로 배치하는것은 C:\anaconda3 혹은 D:\anaconda3 등으로 배치
7. Register Anaconda3 as my default Python
8. 설치 완료 이후 MaC 혹은 Linux 는 배치할 경로에 가서 jupyter notebook 만 치면 끝
   윈도우의 경우엔 Anaconda Prompt 에서 배치할 경로로 이동해서 jupyter notebook 입력  
```
