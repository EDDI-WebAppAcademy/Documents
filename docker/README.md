# Windows 에 Docker 설치하기

```make
1. https://docs.docker.com/desktop/windows/ 이동
2. https://docs.docker.com/desktop/windows/install/ 이동
3. Docker Desktop for Windows 다운로드
4. 제어판 -> 프로그램 설치 및 제거 -> Windows 기능 켜기/끄기 -> Hyper-V 체크
5. 체크 완료 이후 리부팅 (체크 되어 있었다면 리부팅 필요없음)
6. 리부팅 이후 설치 진행
7. 전부 체크
8. Docker 설치 완료 이후 다시 재부팅
9. 나타나는 창에 동의 클릭
10. https://docs.microsoft.com/ko-kr/windows/wsl/install-manual#step-4---download-the-linux-kernel-update-package 패치
11. x64 WSL 업데이트 패치 설치
12. PowerShell 에서 wsl --set-default-version 2 입력
13. 
```
